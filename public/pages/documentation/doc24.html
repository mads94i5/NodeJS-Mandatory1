Reading files

Reading files in Node requires you to import the file system abbreviated as 'fs', using: 

import fs from "fs";

After importing, files can be read either asynchronously or synchronously using: 

// Asynchronous:
fs.readFile(filePath);
// Synchronous:
fs.readFileSync(filePath);

By default this will return a Buffer object. To get the text in the binary data of the Buffer object, you can apply an 'utf-8' parameter or call 'toString()' on the method: 

// utf-8 parameter:
fs.readFile(filePath, 'utf-8');
// toString():
fs.readFile(filePath).toString();
