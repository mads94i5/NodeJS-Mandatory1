Intro to Node.JS

Node is an open source, cross platform runtime environment, where you can run JavaScript.
Can be used for server applications, GUI applications, command line applications etc.

Installation

Download and run the installer from nodejs.org on Windows/macOS/Linux.
Or follow the instructions to install via package manager.

Set up project

Go to github and create a new repository with "Node" gitignore. When using vscode, also add: .vscode
Open a terminal, navigate to the desired project directory using: 

cd <directoryname>

Then clone the github project by writing: 

git clone <url>

Replace <url> with the URL of the github repository.
Once it's finished cloning, simply write: 

code .

This will open the current directory in VSCode.
We can do this, since we will already be in the directory of the project.
Now check the Node version, to make sure everything is working correctly:

node -v

If the version number gets echoed in the terminal, Node is working.
Now, to run JavaScript files with Node, navigate to the directory of the file and type:

node <filename>

You can also open Node REPL (Read-eval-print loop) by simply typing: 

node

REPL can be used to write and execute code directly in the terminal.
For example, if you write: 

2+2

And then press enter, then "4" will be printed to the terminal as well.

Now, to push code changes to the github repository, we first need to add our changes to git:

git add -A

The modifer "-A" means "all", which means all changes will be added to git.
Meanwhile, the modifer "." only adds the current path and all subpaths.
To commit what we added to git, write:

git commit -m "<commit message>"

Then we have committed our local changes to our local git branch.
To push our local git branch to the github repository, type:

git push
